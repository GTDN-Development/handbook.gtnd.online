## ðŸ—‚ Sidebar Structure

Fumadocs builds the sidebar **automatically** based on:

1. **Folders** â†’ become top-level sections
2. **Files inside folders** â†’ become pages
3. **Optional frontmatter** â†’ can control order, titles, and grouping

---

### Example: Frontmatter

```mdx
---
title: "Installation"
order: 2
group: "Getting Started"
---

# Installation

Content goes here...
```

**Frontmatter fields:**

- `title`: What appears in the sidebar
- `order`: Optional numeric value to control the order of items
- `group`: Optional string to group pages under a subsection
- `description`: Optional page description
- `tags`: Optional tags for filtering or search

> Pages without `order` are sorted alphabetically.
