```tsx

```

### Basic example

```tsx

  Copy email
</CopyButton>
```

## Examples

### Display feedback with render prop

Render props are good, and we expose one called `isCopied` within the `children` prop. You can use this pattern to display different content based on the internal state:

<Example>
    <CopyButtonWithRenderProp />
</Example>

```tsx
<CopyButton toCopy="hello@example.com" className="relative">
  {({ isCopied }) => (
    <>
      <span className={cn(!isCopied ? "visible" : "invisible")}>
        hello@example.com
      </span>
      <span
        className={cn(
          "absolute inset-0 flex items-center justify-center",
          isCopied ? "visible" : "invisible"
        )}
      >
        <CheckIcon aria-hidden="true" className="mr-1 size-[1em]" />
        Copied!
      </span>
    </>
  )}
</CopyButton>
```

### useClipboard hook

For more sophisticated implementations, use the `useClipboard` hook directly:

```tsx
import { useClipboard } from "@/components/ui/copy-button";

function MyComponent() {
  const { isCopied, copy } = useClipboard(2000);

  return (
    <div>
      <input id="code" value="npm install my-package" readOnly />
      <button onClick={() => copy(document.getElementById("code").value)}>
        {isCopied ? "Copied!" : "Copy"}
      </button>
    </div>
  );
}
```
